Desenvolvimento Web - CORS

Para testar na web sem problemas de CORS, use uma das seguintes opções:

Opção 1: Chrome com CORS desabilitado

# Feche o Chrome completamente e execute:
google-chrome --disable-web-security --user-data-dir="/tmp/chrome_dev"

# No Mac:
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --disable-web-security --user-data-dir="/tmp/chrome_dev"

Opção 2: Use o emulador mobile (recomendado)

flutter run --dart-define=CITY_NAME="Ouro Preto" --dart-define=FLAVOR=ouroPreto -d emulator-5554

Opção 3: Proxy CORS (desenvolvimento)

Use um proxy CORS como: https://cors-anywhere.herokuapp.com/
